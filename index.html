MOVIE RECOMMENDATION SYSTEM

I created it using collaborative filtering with the help of scikit surprise library, which learns from past user behavior, Basically collaborative filtering is a technique that can filter out items that a user might like on the basis of reactions by similar users. It works by searching a large group of people and finding a smaller set of users with tastes similar to a particular user. It looks for the item that they want and combine them to create a ranked list of suggestions
